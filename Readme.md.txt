# 🎬 Movie Recommender System

This is a content-based movie recommendation system built using **Streamlit**, **Sentence Transformers**, and **cosine similarity**. The app suggests movies similar to a user-entered title based on semantic meaning rather than keywords alone.

---

## 🚀 Features

- 🧠 Uses SentenceTransformer to generate movie embeddings.
- 📊 Computes similarity scores to recommend top 5 most similar movies.
- ✅ Clean and interactive Streamlit UI.
- 🎯 Real-time recommendations with similarity confidence shown as percentages.
